DEPLOYMENT GUIDE

Deploy in 3 Steps

Static cockpit pages + 17 serverless API functions ship together. One vercel --prod and it's live globally.

01
CLONE & INSTALL
Pull the repo, install Vercel CLI. No npm build step needed — static output directory is public/.
# get the project git clone \ github.com/aerosys/9000 cd aerosys-9000 # install vercel CLI npm i -g vercel
02
CONFIGURE ENV
Set your JWT secret and optional environment variables in the Vercel dashboard or via CLI before first deploy.
# set env vars vercel env add \ AEROSYS_JWT_SECRET vercel env add \ AEROSYS_VERSION # link project vercel link
03
DEPLOY
Single command deploys 17 serverless functions + all static pages to Vercel's global edge network.
# deploy to prod vercel --prod # or push via Git git push origin main # auto-deploys on push # verify curl /api/v2/health
ADA NATIVE BINARY (aircraft-side)
# Build release binary gprbuild -P aerosys.gpr \ -Xmode=release -j0 # Run on port 8080 ./bin/aerosys_api 8080
# Coverage build gprbuild -P aerosys.gpr \ -Xmode=coverage # Run tests gnattest -P aerosys.gpr
